# Minecraft Auto Register Login Premium

Provide automatic Register and Login for Offline servers. Also support premium account automatic switch.



## Versions

* "fabricloader": ">=0.18.4",
* "minecraft": "~1.21.8",
* "java": ">=21",
* "fabric-api": "\*"





## Compilation instruction

* Download VS Code
* Add extensions: Java, Extension Pack For Java
* MAJ+CTRL+P => Clean Java Language
* Install Java 21 LTS SDK from JDK Donwloader



* Terminal run : ./gradlew vscode
* Terminal run : ./gradlew clean genSources
* Terminal run : ./gradlew build



You should have sucesseful build \& working autocompletion
